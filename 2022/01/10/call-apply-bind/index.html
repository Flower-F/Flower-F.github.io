this

如果您正在学习 JavaScript，您可能已经了解到了 this 关键字。JavaScript 中的 this 关键字的行为与其他编程语言不同，这给不少程序员带来了困惑。
在其他面向对象编程语言中，this 关键字总是指向类的当前实例。而在 JavaScript 中，它的值取决于函数的调用方式。
我们来通过一些例子说明一下 JavaScript 中的 this。
