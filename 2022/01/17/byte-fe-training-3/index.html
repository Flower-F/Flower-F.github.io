浏览器内核

  • IE：Trident
  • Firefox：Gecko
  • Chrome：Webkit / Blink
  • Safari：Webkit
  • Opera：Presto

Web 标准组织

  • W3C: HTML、XML、CSS、DOM、Web API 等
  • ECMA: ECMAScript
  • WHATWG: 与 W3C 合作制定 HTML 和 DOM
  • ITEF: TCP/IP、FTP 等网络协议

console

  • console.warn
  • console.error
  • console.table：格式化打印数组 / JSON 数据
  • console.dir：按照树形结构打印 DOM 节点
