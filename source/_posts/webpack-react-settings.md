# 初始化

先初始化项目

```bash
npm init
```

运行命令初始化 package.json 文件。

安装 webpack 和 webpack-cli。

```bash
yarn add webpack webpack-cli -D
```

安装 react。

```bash
yarn add react react-dom
```

新建文件 webpack.config.js，书写配置信息。
