题目链接：

https://leetcode-cn.com/problems/climbing-stairs/

解法：

f(i) 表示到第 n 阶楼梯一共有多少种解法

dp：f(i) = f(i - 1) + f(i - 2)，然后再通过变量存储来降低空间复杂度到 O(1) 即可。

```js
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    if(n === 1) return 1;
    if(n === 2) return 2;
    let ans;
    pre1 = 1, pre2 = 2
    for(let i = 3; i <=n; i++) {
        ans = pre1 + pre2;
        pre1 = pre2;
        pre2 = ans;
    }
    return ans;
};
```
